# Lign Financial

A new Flutter project for Lign Financial.

## Getting Started

This project uses a Feature-first architecture with Riverpod for state management and GoRouter for navigation.

### Prerequisites

- Flutter SDK
- Dart SDK

### Running the App

```bash
flutter pub get
flutter run
```

### Running Tests

```bash
flutter test
```

### Architecture

The project is structured as follows:

- `lib/src/features`: Feature-based modules.
- `lib/src/common_widgets`: Shared UI components.
- `lib/src/constants`: App-wide constants.
- `lib/src/routing`: Navigation configuration.
- `lib/src/utils`: Helper functions.

### State Management

This project uses `flutter_riverpod` for state management.
